De-Time Method Gallery

Explore De-Time's decomposition methods. Each card links to a dedicated page with executable code, output, and a decomposition plot. All examples use compact synthetic data and run quickly.

Tip
Regenerate all assets with python scripts/generate_notebook_gallery.py. Jump to downloads for the full notebook, Python source, or zipped example.
