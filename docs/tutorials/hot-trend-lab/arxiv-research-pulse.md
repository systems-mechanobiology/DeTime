# arXiv Research Pulse

arXiv is the strongest starting point for this column because it turns AI research hype into a measurable public time series.

## Why it can travel

| Reason | Explanation |
|---|---|
| Public data | arXiv exposes an API and public statistics pages. |
| High interest | AI, agents, coding agents, mechanistic interpretability, and AI-for-science are global topics. |
| Natural cycles | Submissions have weekly, monthly, and conference-deadline rhythms. |
| Safe framing | It is a research-output pulse, not an investment recommendation. |

## Core questions

| Question | De-Time component |
|---|---|
| Which categories are structurally growing? | `trend` |
| Which categories are deadline-driven? | `season` |
| Which categories had unusual bursts? | `residual` |
| Which topic deserves a column this week? | residual-shock rank plus trend score |
